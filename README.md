# CSGO_Plugin
一个在网页上展示你的CSGO相关信息的插件
## 使用方法
```html
<style> 
.csgo-stat-box{ 
    display:block; 
    max-width:500px; 
    height:220px; 
    border-radius:4px; 
} 
</style> 
<iframe class="csgo-stat-box" src="https://api.learningman.top/csgo/plugin?steamid={}&rankid={}" frameborder="0"></iframe>
```
